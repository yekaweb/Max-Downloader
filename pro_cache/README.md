# 🧠 Pro Cache - سیستم کش هوشمند حرفه‌ای برای Max-Downloader

## 📋 معرفی پروژه

سیستم Pro Cache یک زیرسیستم پیشرفته برای ربات Max-Downloader است که با استفاده از تکنیک‌های کش هوشمند، سرعت پاسخگویی به درخواست‌های تکراری را از چند دقیقه به کمتر از یک ثانیه کاهش می‌دهد.

### 🎯 اهداف اصلی

1. **کاهش 99% زمان پردازش** برای لینک‌های تکراری
2. **کاهش بار سرور** و مصرف پهنای باند
3. **بهبود تجربه کاربری** با ارائه گزینه‌های هوشمند
4. **ذخیره چندین کیفیت** از هر محتوا
5. **مدیریت هوشمند حافظه** و پاکسازی خودکار

### 🔑 ویژگی‌های کلیدی

- **تشخیص لینک تکراری** با الگوریتم SHA-256
- **ذخیره File ID تلگرام** برای ارسال سریع
- **پشتیبانی از چندین کیفیت** برای هر محتوا
- **رابط کاربری زیبا** با دکمه‌های inline
- **مدیریت TTL** (Time To Live) برای کش‌ها
- **استراتژی LRU** برای پاکسازی حافظه

### 🏗️ معماری سیستم

```
┌─────────────────────────────────────────────────────┐
│                   User Request                      │
│                        ↓                            │
│              ┌─────────────────┐                   │
│              │ URL Hash Engine │                   │
│              └────────┬────────┘                   │
│                       ↓                            │
│            ┌──────────────────────┐                │
│            │ Cache Lookup Service │                │
│            └──────────┬───────────┘                │
│                       ↓                            │
│         ┌─────────────┴──────────────┐             │
│         ↓                            ↓             │
│    Cache Hit                    Cache Miss         │
│         ↓                            ↓             │
│ ┌──────────────┐           ┌──────────────┐       │
│ │ Quick Return │           │ Fresh Download│       │
│ │   Service    │           │    Service    │       │
│ └──────────────┘           └──────────────┘       │
│         ↓                            ↓             │
│    Send Cached                 Download New        │
│      File ID                   Store in Cache      │
└─────────────────────────────────────────────────────┘
```
