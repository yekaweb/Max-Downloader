# 🗺️ Mobile App Implementation Roadmap

این نقشه راه برای توسعه اپلیکیشن اندروید Max Downloader با استفاده از **Kotlin + Jetpack Compose** و موتور `yt-dlp` تنظیم شده است.

## فاز ۱: زیرساخت و طراحی اولیه (Foundation & UI)
- [ ] راه‌اندازی پروژه اندروید در Android Studio.
- [ ] کانفیگ Jetpack Compose.
- [ ] ساخت سیستم رنگی و کامپوننت‌های پایه نئومورفیسم (Neumorphism Buttons, Cards, TextFields).
- [ ] طراحی صفحه اصلی (فیلد جستجو و دکمه دانلود).
- [ ] طراحی صفحه تنظیمات.

## فاز ۲: موتور استخراج اطلاعات (Extraction Engine)
- [ ] ایمپورت کردن لایبرری `youtubedl-android` (یا پورت کردن کدهای پایتونی با Chaquopy).
- [ ] ارسال لینک از UI به موتور دانلود و دریافت JSON متادیتا.
- [ ] پارس کردن اطلاعات (مدت زمان، کیفیت‌ها، کدک‌ها، عکس بندانگشتی).
- [ ] طراحی دیالوگ کشویی (BottomSheet) برای انتخاب کیفیت‌ها دقیقاً مشابه ربات.

## فاز ۳: موتور دانلود و ترکیب (Download & Muxing)
- [ ] راه‌اندازی `FFmpeg` اندروید برای ترکیب ویدیوهای فاقد صدا (1080p به بالا) با فایل‌های صوتی.
- [ ] پیاده‌سازی سرویس پس‌زمینه (Foreground Service) برای ادامه دانلود هنگام بسته بودن برنامه.
- [ ] نمایش پروگرس‌بار زنده روی نوتیفیکیشن گوشی و داخل خود اپلیکیشن.
- [ ] مدیریت خطاها و Resume کردن دانلودهای قطع شده.

## فاز ۴: امکانات جانبی و گالری
- [ ] ذخیره متادیتای دانلودها در دیتابیس Room (تاریخچه).
- [ ] ساخت تب "دانلودهای من" برای مدیریت، پخش و اشتراک‌گذاری فایل‌ها.
- [ ] پشتیبانی از دانلود مستقیم زیرنویس‌ها.

## فاز ۵: درآمدزایی و انتشار (Monetization)
- [ ] پیاده‌سازی سیستم تبلیغات (AdMob / Tapsell) برای کاربران رایگان.
- [ ] پیاده‌سازی خرید درون برنامه‌ای (In-App Billing) برای ارتقا به نسخه پرو.
- [ ] قفل کردن کیفیت‌های 4K و قابلیت پلی‌لیست برای نسخه پرو.
- [ ] خروجی گرفتن نهایی (APK/AAB) و انتشار.
