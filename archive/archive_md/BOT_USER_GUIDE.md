# 📱 راهنمای استفاده از DLBot - برای کاربران عادی

---

## 🎯 قابلیت‌های ربات

ربات DLBot می‌تواند ویدیو‌ها و محتوا را از پلتفرم‌های زیر دانلود کند:

| پلتفرم | قابلیت |
|--------|--------|
| 🎥 **YouTube** | دانلود ویدیو، موسیقی، لیست‌های پخش |
| 📸 **Instagram** | دانلود پست، ریل، استوری |
| 🐦 **Twitter (X)** | دانلود ویدیو و توییت‌ها |

---

## ⚡ شروع سریع

### گام 1: دستور `/start`

```
/start
```

**پاسخ ربات:**
```
🤖 سلام به DLBot!

من ربات دانلود حرفه‌ای برای:
• 🎥 YouTube
• 📸 Instagram
• 🐦 Twitter

برای شروع دکمه پایین را بزن!
```

### گام 2: انتخاب "📥 دانلود ویدیو"

دکمه **📥 دانلود ویدیو** را بزنید

### گام 3: انتخاب پلتفرم

**کدام یکی را دانلود می‌خواهید؟**
- 🎥 YouTube
- 📸 Instagram
- 🐦 Twitter

### گام 4: ارسال لینک

**لینک ویدیو را کپی و ارسال کنید:**

```
https://youtu.be/...
```

یا برای Instagram:
```
https://instagram.com/p/...
```

### گام 5: دریافت ویدیو

⏳ **درحال دانلود...**
↓
📤 **درحال آپلود...**
↓
✅ **ویدیو دریافت شد!**

## 🎬 مثال‌های عملی

### دانلود ویدیو YouTube

```
1. /download را ارسال کن
2. روی "🎥 YouTube" بزن
3. لینک YouTube را ارسال کن:
   https://youtu.be/UFcgD-tcyrs
4. منتظر دانلود باشید ✅
```

### دانلود ریل Instagram

```
1. /download را ارسال کن
2. روی "📸 Instagram" بزن
3. لینک ریل را ارسال کن:
   https://instagram.com/reel/ABC123/
4. فایل دریافت کنید ✅
```

### دانلود ویدیو Twitter

```
1. /download را ارسال کن
2. روی "🐦 Twitter" بزن
3. لینک توییت را ارسال کن:
   https://twitter.com/user/status/123456
4. ویدیو دریافت شود ✅
```

## 🔧 نکات مهم

### ✅ لینک‌های معتبر

```
✅ صحیح:
- https://youtu.be/ABC123
- https://www.youtube.com/watch?v=ABC123
- https://instagram.com/p/ABC123
- https://twitter.com/user/status/123456

❌ غلط:
- یوتیوب
- ویدیو
- فقط رقم‌ها (ABC123)
```

### ⚠️ محدودیت‌ها

```
❌ نمی‌توان دانلود کرد:
- ویدیوهای Private یا محدود
- محتوای پنهان یا حذف‌شده
- ویدیوهای بسیار بزرگ (>4GB)
- ویدیوهایی که صاحب آن‌ها اجازه نداده‌اند

⏱️ مدت زمان:
- دانلود معمولا 1-5 دقیقه طول می‌کشد
- ویدیوهای بزرگ تر بیشتر وقت نیاز دارند
```

### 🎯 کیفیت‌های دستیاب

```
🎥 YouTube:
- 480p (480 پیکسل)
- 720p (کیفیت معمول)
- 1080p (کیفیت بالا)
- 4K (کیفیت بسیار بالا)

📸 Instagram:
- عکس: 1080×1080
- ریل: 720p یا 1080p

🐦 Twitter:
- 720p یا 1080p
```

```
  - کیفیت پایین‌تری انتخاب کنید
  - ویدیوی کوتاه‌تری دانلود کنید
```

## 💡 نکات مفید

### 📌 نوک و ترفندها

```
1️⃣ لینک را مستقیماً کپی/پیست کنید
   (بدون جملات اضافی)

2️⃣ صبر کنید تا دانلود کامل شود
   (پیام "⏳ درحال دانلود..." را می‌بینید)

3️⃣ اگر ربات شلوغ است، کمی بعد تلاش کنید
   (امروز ممکن است کند باشد)

4️⃣ فایل‌های دانلود‌شده 48 ساعت نگاه داشته می‌شوند
   (بعد از این مدت حذف می‌شوند)
```
