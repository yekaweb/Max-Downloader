## 📥 مرحله 1: نصب برنامه‌ها

### 1.1 - نصب Python

**چرا؟** ربات با زبان Python نوشته شده است.

**نحوه نصب** (ویندوز):
1. برو به **www.python.org**
2. روی **Downloads** کلیک کن
3. دانلود **Python 3.11** یا جدیدتر
4. دبل‌کلیک روی فایل دانلود شده
5. **✅ ✔️ Add Python to PATH** را تیک بزن (خیلی مهم!)
6. **Install Now** را کلیک کن
7. صبر کن تا تمام شود

**تست کردن**:
```
Windows Key + R کلیک کن
cmd وارد کن و Enter بزن
python --version وارد کن
```
اگر نسخه نشان داد = ✅ درست است

### 1.2 - نصب Git

**چرا؟** برای دانلود کد پروژه از اینترنت

**نحوه نصب** (ویندوز):
1. برو به **www.git-scm.com**
2. **Download** را کلیک کن (خودکار ویندوز انتخاب می‌شود)
3. فایل دانلود شده را دبل‌کلیک کن
4. **Next > Next > Install** کلیک کن
5. صبر کن تا تمام شود

## 🔐 مرحله 2: دریافت توکن ربات

### 2.1 - ایجاد ربات در تلگرام

**مراحل**:

1. **تلگرام را باز کن**
   - گوشی‌ات را بردار
   - اپلیکیشن تلگرام را باز کن

2. **به BotFather بروز**
   - در جستجو جستجو کن: `@BotFather`
   - روی **@BotFather** کلیک کن (علامت ✓ دارد)

3. **فرمان /start وارد کن**
   - روی `/start` کلیک کن
   - پیام‌ها نشان می‌دهد

4. **فرمان /newbot وارد کن**
   - روی `/newbot` کلیک کن
   - منتظر بمان

5. **اسم ربات را وارد کن**
   - چیزی مثل: `MyDownloaderBot` یا `MyVideoBot`
   - و یا هر نام دیگری که دوست دارید
   - Enter بزنید

6. **نام کاربری ربات را وارد کن**
   - باید با `bot` تمام شود
   - مثل: `my_downloader_bot` یا `video_dl_bot`
   - نام کاربری باید منحصر‌به‌فرد باشد
   - Enter بزنید

7. **توکن را کپی کنید**
   - یک توکن بلند نشان می‌دهد (مثل):
   ```
   123456789:ABCDEFGhijklmnopqrstuvwxyz-123456789
   ```
   - **روی آن کلیک کنید یا آن را انتخاب و Ctrl+C کنید**
   - این توکن را **جایی ذخیره کنید** (یادداشت یا فایل)

### 2.2 - دریافت App ID و App Hash

**چرا؟** برای کنترل بیشتر روی حساب تلگرام

**مراحل**:

1. **مرورگر باز کن**
   - Google Chrome یا Firefox باز کن

2. **به my.telegram.org برو**
   - آدرس را وارد کن: `https://my.telegram.org`

3. **Phone number را وارد کن**
   - شماره تلگرام‌تان را با کد کشور وارد کنید
   - مثال: `+989123456789` (ایران)

4. **کد تایید را وارد کن**
   - کد تایید به تلگرام‌تان می‌رسد
   - آن را وارد کنید

5. **API development tools کلیک کن**
   - یک صفحه جدید می‌آید

6. **App دیگری ایجاد کن** (اگر نیست)
   - **Create new application** کلیک کن
   - اطلاعات را پر کنید:
     - App title: `Downloader Bot` یا هر چیز دیگر
     - Short name: `downloader_bot`
   - **Create Application** کلیک کن

7. **App ID و App Hash را کپی کنید**
   ```
   api_id: 123456
   api_hash: abcdef1234567890abcdef1234567890
   ```
   - **هر دو را جایی ذخیره کنید**

## 📁 مرحله 3: تنظیم پوشه پروژه

### 3.1 - پوشه پروژه را آماده کنید

**مراحل**:

1. **پوشه‌ای برای پروژه ایجاد کنید**
   - کلیک راست روی دسکتاپ
   - **New Folder** را کلیک کنید
   - نام دهید: `DLBot` یا `DownloaderBot`

2. **VS Code را باز کنید**
   - **File > Open Folder** را کلیک کنید
   - پوشه‌ای را که ساختید انتخاب کنید
   - **Select Folder** را کلیک کنید

3. **فایل `.env` ایجاد کنید**
   - VS Code میں **File > New File** کلیک کنید
   - نام دهید: `.env` (دقیقاً همین)
   - این محتوی را پیست کنید:

```
# تلگرام
TELEGRAM_BOT_TOKEN=توکن_شما_اینجا
TELEGRAM_APP_ID=app_id_شما
TELEGRAM_APP_HASH=app_hash_شما

# Admin ID (بعداً)
ADMIN_ID=آیدی_تلگرام_شما

# دیتابیس (برای ابتدا می‌توانید SQLite استفاده کنید)
DATABASE_URL=sqlite:///dlbot.db

# در تولید، از PostgreSQL استفاده کنید:
# DATABASE_URL=postgresql://user:password@localhost/dlbot
```

4. **توکن و ID‌ها را جایگزین کنید**
   - **TELEGRAM_BOT_TOKEN**: توکنی که از BotFather گرفتید
   - **TELEGRAM_APP_ID**: App ID شما
   - **TELEGRAM_APP_HASH**: App Hash شما
   - **ADMIN_ID**: آیدی تلگرام شما (اگر نمی‌دانید، بعداً کپی کنید)

**مثال:**
```
TELEGRAM_BOT_TOKEN=1234567890:ABCDEFGhijklmnopqrstuvwxyz
TELEGRAM_APP_ID=123456
TELEGRAM_APP_HASH=abcdef1234567890abcdef1234567890
ADMIN_ID=987654321
DATABASE_URL=sqlite:///dlbot.db
```

### 3.2 - Admin ID خود را پیدا کنید

اگر Admin ID خود را نمی‌دانید:

1. **به @userinfobot بروید**
   - در تلگرام جستجو کنید: `@userinfobot`
   - روی `/start` کلیک کنید
   - آیدی شما نشان می‌دهد
   - کپی کنید و در `.env` قرار دهید

## 💾 مرحله 4: راه‌اندازی دیتابیس

### 4.1 - Terminal را باز کنید

**مراحل**:

1. **VS Code میں Terminal را باز کنید**
   - **View > Terminal** کلیک کنید
   - یا `Ctrl + ~` کلیک کنید

2. **مطمئن شوید در پوشه پروژه هستید**
   - خط آخر باید نشان دهد:
   ```
   C:\Users\YourName\Desktop\DLBot>
   ```

### 4.2 - برنامه‌های Python را نصب کنید

**در Terminal کپی و پیست کنید**:

```bash
pip install -r requirements.txt
```

**چه می‌کند؟** تمام برنامه‌های لازم را نصب می‌کند

**انتظار بکشید** (شاید ۵-۱۰ دقیقه)

اگر خطا داد:
```
python -m pip install --upgrade pip
```

سپس دوباره تلاش کنید.

### 4.3 - دیتابیس را ایجاد کنید

**در Terminal کپی و پیست کنید**:

```bash
alembic upgrade head
```

**چه می‌کند؟** یک دیتابیس خالی برای ذخیره‌سازی اطلاعات ایجاد می‌کند

## ❌ حل مشکلات

### مشکل 1: "Python not found"

**خطا:**
```
'python' is not recognized as an internal or external command
```

**حل:**
1. Python را دوباره نصب کنید
2. **حتماً "Add to PATH" را انتخاب کنید**
3. کامپیوتر را دوباره شروع کنید

---

### مشکل 2: "Module not found"

**خطا:**
```
ModuleNotFoundError: No module named 'yt_dlp'
```

**حل:**
```bash
pip install -r requirements.txt
```

دوباره تلاش کنید.

---

### مشکل 3: ربات پاسخ نمی‌دهد

**عللی احتمالی:**
1. توکن اشتباه است
2. API Hash اشتباه است
3. App ID اشتباه است

**حل:**
1. فایل `.env` را دوباره بررسی کنید
2. توکن، App ID و App Hash را دوباره کپی و پیست کنید
3. Terminal را بند کنید و دوباره شروع کنید

---

### مشکل 4: "Cannot connect to database"

**خطا:**
```
sqlalchemy.exc.OperationalError
```

**حل:**
```bash
alembic upgrade head
```

دوباره اجرا کنید.

### مشکل 5: "Permission denied"
