## 🎓 مقدمه

این ربات یک **برنامه است** که:
- ✅ روی کامپیوتر شما کار می‌کند
- ✅ از تلگرام فرمان می‌گیرد
- ✅ ویدیو دانلود می‌کند

---

## 🖱️ چیزهایی که باید باز کنید

**۲ تا برنامه لازم است:**

### ۱️⃣ **VS Code** (ویرایشگر کد)
- فایل‌ها را می‌بندیم و می‌کنیم

### ۲️⃣ **Terminal** (خط فرمان)
- دستورات را می‌زنیم

**اگر ندارید:**
- VS Code: code.visualstudio.com
- Terminal: هم‌کاری Windows است (Press Windows+R, type cmd)

---

## 🚀 قدم ۱ تا ۱۰:

### **قدم ۱: پوشه پروژه را باز کنید**

✋ **STOP** - دقیق دنبال کنید!

1. **Windows Explorer را باز کنید** (فایل منیجر)
   - میانبر: `Windows + E`

2. **به این پوشه برو:**
   ```
   d:\telgram bot md backup 2- Copy\
   ```

3. **روی پوشه راست کلیک کنید**

4. **"Open with Code" کلیک کنید** (اگر ندید، باید VS Code نصب کنید)

**نتیجه**: VS Code باز می‌شود

---

### **قدم ۲: Terminal را باز کنید (درون VS Code)**

VS Code میں:

1. بالای صفحه: `View > Terminal`
2. یا میانبر: `Ctrl + ~`

**نتیجه:** پایین VS Code، یک جعبه سیاه ظاهر می‌شود (Terminal)

---

### **قدم ۳: چک کنید مسیر درست است**

در Terminal نشان دهد:
```
d:\telgram bot md backup 2- Copy>
```

اگر ندید:
```bash
cd d:\telgram\ bot\ md\ backup\ 2-\ Copy
```

Enter بزنید.

## 🛑 اگر مشکل داشت:

### ❌ "pip: command not found"

**معنی**: Python نصب نیست

**حل**:
1. python.org برو
2. Python نصب کن
3. **مهم**: "Add to PATH" تیک بزن!
4. کامپیوتر دوباره شروع کن
5. دوباره تلاش کن

---

### ❌ "Module not found"

**معنی**: برنامه‌ها نصب نشدند

**حل**:
```bash
pip install -r requirements.txt
```

**دوباره تلاش کن**

---

### ❌ "Bot token invalid"

**معنی**: توکن اشتباه است

**حل**:
1. فایل `.env` را باز کنید
2. `BOT_TOKEN` را بررسی کنید (خط ۴)
3. درست است؟
4. ربات را متوقف کنید (Ctrl+C)
5. دوباره ران کنید

---

### ❌ ربات جواب نمی‌دهد

**معنی**: ربات کار می‌کند اما تلگرام برقرار نمی‌کند

**حل**:
1. اینترنت بررسی کنید
2. ربات را متوقف کنید (Ctrl+C)
3. دوباره ران کنید
4. صبر ۳۰ ثانیه کنید
5. دوباره تلاش کنید
